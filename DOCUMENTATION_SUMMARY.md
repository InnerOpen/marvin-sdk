## 🚀 How to Use

### Development

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally
npm run docs:serve

# Visit http://localhost:8000
```

### Build

```bash
# Build static site
npm run docs:build

# Output: site/ directory
```

### Deploy

```bash
# Deploy to GitHub Pages
npm run docs:deploy

# Or push to main - GitHub Actions auto-deploys
git push origin main
```
