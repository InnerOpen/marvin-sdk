## 🚀 Quick Start

### Install Dependencies

```bash
# Python dependencies
pip install -r requirements.txt

# npm dependencies (already installed)
npm install
```

### Serve Locally

```bash
npm run docs:serve
# or
mkdocs serve
```

Visit **http://localhost:8000**

### Build Static Site

```bash
npm run docs:build
# or
mkdocs build
```

Output: `site/` directory

### Deploy to GitHub Pages

```bash
npm run docs:deploy
# or
mkdocs gh-deploy
```

Or push to `main` branch - GitHub Actions will auto-deploy.
